# 자유와혁신 공식 웹사이트

🇰🇷 자유와혁신의 공식 웹사이트입니다.

## 📋 사이트 구조

```
party-website/
├── admin/                 # CMS 관리자 페이지
├── content/              # 컨텐츠 저장소
│   ├── notices/         # 공지사항
│   ├── news/           # 소식/활동
│   ├── events/         # 행사/집회
│   ├── policies/       # 정책
│   ├── members/        # 주요 인물
│   └── settings/       # 사이트 설정
├── images/             # 이미지 파일
├── index.html          # 메인 페이지
├── about.html          # 당 소개
├── policy.html         # 정책
├── news.html           # 소식
├── members.html        # 당원
├── support.html        # 참여/후원
└── resources.html      # 자료실
```

## 🔧 관리자 기능 사용법

### 1. CMS 접속 방법
1. 웹사이트 주소 뒤에 `/admin` 추가
   - 예: `https://jayu.netlify.app/admin`
2. Netlify 계정으로 로그인
3. 관리자 대시보드 접속

### 2. 컨텐츠 관리

#### 📢 공지사항 작성
1. 관리자 → "공지사항" 클릭
2. "새 공지사항" 버튼 클릭
3. 필수 정보 입력:
   - 제목
   - 발행일
   - 중요도 (일반/중요/긴급)
   - 카테고리 (공지/보도자료/성명서/안내)
   - 내용 (마크다운 지원)
4. "게시" 버튼으로 발행

#### 📅 행사/집회 등록
1. 관리자 → "행사/집회" 클릭
2. "새 행사" 버튼 클릭
3. 필수 정보 입력:
   - 행사명
   - 일시
   - 장소
   - 주최
   - 연락처
   - 유형 (집회/토론회/간담회/행진/세미나)
   - 지역
4. 선택사항:
   - 포스터 이미지 업로드
   - 상세 설명
   - 참가비
   - 사전신청 여부

#### 📝 정책 관리
1. 관리자 → "정책" 클릭
2. "새 정책" 버튼 클릭
3. 정보 입력:
   - 정책명
   - 분야 (경제/일자리, 교육/복지, 안보/외교 등)
   - 우선순위 (핵심/중요/일반)
   - 요약
   - 배경 설명
   - 목표
   - 세부내용
   - 기대효과

#### 👥 인물 정보 관리
1. 관리자 → "주요 인물" 클릭
2. "새 인물" 추가
3. 정보 입력:
   - 이름, 직책
   - 프로필 사진
   - 연락처, 이메일
   - 약력
   - 주요 경력
   - 학력

### 3. 이미지 업로드
1. CMS에서 이미지 필드 클릭
2. "미디어 라이브러리"에서 선택 또는 새 이미지 업로드
3. 자동으로 `/images` 폴더에 저장됨

### 4. 사이트 기본 설정
1. 관리자 → "사이트 설정" 클릭
2. 기본 정보 수정:
   - 당명, 슬로건
   - 연락처, 이메일, 주소
   - 소셜미디어 링크
   - 당원 가입 링크
   - 후원 링크

## 🚀 배포 방법

### Netlify 자동 배포
1. GitHub에 코드 push
2. Netlify가 자동으로 감지하여 배포
3. 보통 1-2분 내 사이트 업데이트 완료

### 수동 배포 (필요시)
1. Netlify 대시보드 접속
2. "Deploys" 탭 → "Trigger deploy" 클릭
